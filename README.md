# Finance Management System

## Project Overview
Finance Management System (Admin Panel) developed for Lending Authorities using React Native and Firebase.

The system helps manage:
- Customer records
- Loan applications
- Payment tracking
- Reports and analytics
- Authentication and security

## Technology Stack
- Frontend: React Native
- Backend & Database: Firebase

## Features
### Dashboard
- Loan statistics
- Active users overview
- Financial summaries

### Customer Management
- Add/Edit/Delete customers
- KYC information storage

### Loan Management
- Manage loan applications
- Loan approval tracking

### Payment Tracking
- EMI/payment records
- Payment history management

### Reports
- Financial reports
- Loan analytics

### Authentication
- Secure admin login
- Firebase authentication
